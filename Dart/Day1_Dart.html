أهلاً بك في Flutter

ما هو Flutter؟

هو UI Toolkit من جوجل، بيسمحلك تبني تطبيقات جميلة وعالية الأداء لأي شاشة (Android, iOS, Web, Desktop) من **قاعدة كود واحدة**.

🚀 أداء أصلي (Native Performance)

فلاتر مش "web wrapper"، فلاتر بيرسم مباشرة بـ Skia Engine.

🔥 Hot Reload

عدل في الكود وشوف النتيجة في أقل من ثانية بدون Restart.

🎨 كل شيء هو Widget

تصميم الواجهات بقى زي تركيب قطع الـ Lego.

📊

مقارنة تفصيلية: Native vs Flutter

المعيار Native (أصيل) Flutter (عابر للمنصات)
عدد الأكواد (Codebase) كود منفصل لكل نظام كود واحد للكل
لغات البرمجة Kotlin / Swift Dart
سرعة التطوير أبطأ (تحتاج تكرار العمل) أسرع بـ 2x على الأقل
الأداء (Performance) الأقصى (100%) قريب جداً (99.9%)
التكلفة (Budget) عالية (تحتاج فريقين) اقتصادية (تحتاج مبرمج فلاتر واحد)

المحرك الخفي: لغة Dart

جوجل اختارت Dart لـ Flutter لأسباب تقنية ذكية جداً:

  • ✔ مزدوجة الترجمة (JIT & AOT): بتقدم سرعة في التطوير (Warm up) وسرعة في الإنتاج (Runtime).
  • ✔ Null Safety: لغة آمنة جداً بتقلل أخطاء "التطبيق قفل فجأة".
  • ✔ Object Oriented: منظمة وبتخلي الكود "نظيف" وقابل لإعادة الاستخدام.

تجهيز بيئة العمل (SDK)

إزاي نبدأ كود؟

1. Dart SDK

حمل شنطة الأدوات اللي بتفهم كود Dart.

choco install dart-sdk

2. DartPad

جرب الكود فوراً في المتصفح بدون تثبيت أي شيء.

dartpad.dev

ما هو الـ SDK؟

اختصار لـ **Software Development Kit**. هو المترجم والأدوات اللي بتحول الكود اللي كتبته لبرنامج شغال فعلياً.

🚀

تثبيت Dart SDK خطوة بخطوة

1. استخدام Chocolatey (الطريقة الأسهل)

تأكد من فتح PowerShell كمسؤول (Administrator)، ثم نفذ الآتي:

* لو Chocolatey مش مثبت عندك، اتبع الخطوات دي بالظبط:

  1. ابحث عن PowerShell في قائمة Start.
  2. اضغط كليك يمين واختار Run as Administrator.
  3. انسخ الكود اللي تحت وصقّه في الـ PowerShell واضغط Enter:
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

بعد ما يخلص، اقفل الـ PowerShell وافتحه تاني كمسؤول، واكتب أمر تثبيت الـ SDK:

# تثبيت Dart SDK
choco install dart-sdk

2. إعداد متغيرات البيئة (Environment Variables)

عادةً Choco بيقوم بالمهمة دي، لكن لو حبيت تتأكد أو تعملها يدوي:

  1. افتح قائمة Start وابحث عن "Edit the system environment variables".
  2. اضغط على زر Environment Variables في الأسفل.
  3. في خانة System Variables (اللي تحت)، ابحث عن Path واضغط Edit.
  4. اضغط New وأضف مسار الـ bin (غالباً: C:\tools\dart-sdk\bin).
  5. اضغط OK واقفل أي Terminal مفتوح وافتحه تاني عشان يحس بالتغيير.

تأكيد النجاح (Verification):

افتح الـ Terminal واكتب:

dart --version

لو ظهر لك رقم الإصدار، يبقى مبروك! أول خطوة في رحلتك كملت بنجاح. 🎉

Null Safety: درع الحماية

"في اللغات القديمة، لو المتغير 'فاضي' وحاولت تستخدمه التطبيق بيفجر (Crash). دارت اخترعت الحل عشان تحمي تطبيقاتك."

علامة الاستفهام (؟)

int age = 20; // مستحيل يكون Null
int? score; // ممكن يكون Null (فاضي)

if (score != null) {
  print(score + 10);
}

Null Safety بتجبرك تفكر: "هل المتغير ده مسموح يكون فاضي؟" وده بيمنع 70% من أخطاء المبتدئين.

الـ Switch: ملك القوائم (Menu Master)

"تخيل إنك بتبني نظام لشركة شحن. لو الشحنة (عادية) التكلفة 50، لو (سريعة) التكلفة 100، لو (دولية) التكلفة 200... بدل ما تستخدم 100 'if'، الـ Switch بتسهل الموضوع."

ليه نستخدم Switch؟

  • الوضوح: الكود بيبقى أنضف وأسهل في القراءة.
  • الأداء: في بعض الحالات بيبقى أسرع من الـ nested if.
  • الدقة: بتجبرك توضح كل حالة لوحدها.
String orderType = "Express";

switch (orderType) {
  case "Normal":
    print("Cost: 50 EGP");
    break; // لازم تخرج من السويتش بعد التنفيذ
  case "Express":
    print("Cost: 100 EGP");
    break;
  default: // حالة الأمان (لو مدخلش في أي كيس)
    print("Invalid Type");
}

دمج النصوص واستقبال البيانات

1. String Interpolation (الدمج الذكي)

بدل ما نستخدم الـ + الكتير، بنستخدم علامة $:

String name = "Ali";
int items = 5;
// الطريقة العادية (Concatenation)
print("Hello " + name + ", you have " + items.toString() + " items.");

// الطريقة الذكية (Interpolation)
print("Hello $name, you have $items items.");
print("Total: ${items * 10} EGP"); // لو فيه عملية حسابية بنستخدم {}

2. User Input (استقبال البيانات)

عشان التطبيق "يتكلم" مع المستخدم، بنحتاج مكتبة dart:io:

import 'dart:io';

void main() {
  stdout.write("Enter your name: ");
  String? input = stdin.readLineSync();
  print("Welcome $input!");
}

احسب الإجمالي. طبق خصم 15% لو المنتج "juice" أو "milk" والكمية > 10. استخدم String Interpolation لطباعة الفاتورة بشكل جذاب.
